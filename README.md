# README

# Tagipedia Task

Task Requirements:
- 1- Search REST API.
- 2- DB that have products table with one column product name.
- 3- Unit testing.

# Prerequisites:
* Ruby version '2.5.1'
* Rails version '5.2.2'
* Database 'Sqlite3'

# Setup & installation:
- Clone the app
- Install bundles `bundle install`
- Run `rails db:create`
- Run `rails db:migrate`
- Run `rails db:seed`
- Run server `rails s`
- Use Postman or whatever the tool to test the api.
- Search endpoint is `/api/v1/products/search_product?name="Prodcut name"`
